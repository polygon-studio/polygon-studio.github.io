Projects

Here's a sample of some of my projects

Saturn Polygon Tool

This tool is designed to help with the development of homebrew on the Sega Saturn. The unique architecture of the console, particularly the VDP 1 and 2 GPUs provides a unique challenge for bringing 3D graphics the Saturn. This tool was written in Python and works the with Blender to convert 3D geometry into textured sprite quads that can be used in Saturn development with the official SGL SDK or the open source Jo Engine.


Pugi

Pugi is one of the first projects I developed for the mobile platform. It's a marble madness style game where the player must get the titular character Pugi to the end of each stage as quickly as possible picking up stars for points and hourglasses to extend time. The game itself will never end however, that's because it uses a procedural level generation algorithm to endlessly create obstacle courses.


Hammerkid Pocket

Hammerkid Pocket is a side game of one of my pet projects, this builds upon the endless structure of Pugi with a more advanced level generation algorithm. The levels created by the algorithm are more complex and allow for paths to overlap each other while still being able to create consistently completable levels. The hills and islands created by the algorithm allow for more interesting levels that the player can fully explore utilizing full 3D character control.



About

I'm a programmer and designer with a passion for creating interactive media and computer graphics.

My 3D journey started with modding games using 3D model creation tools like Anim8tor, Wings3D, 3D Canvas, Milkshape and Gmax. Once my desire to create models evolved into a desire to bring them to life I delved into the world of programming. I started with Dark Basic before moving on to C wading my way through countless 3D engines during the great engine wars of the mid 2000's before Unity and UDK appeared and cleaned house.

I have experience developing not only for computers, but have also released products on mobile and worked with propietary hardware like video game consoles.
